链管理模块用于管理系统支持的区块链网络配置。每条链包含名称、RPC 节点和提币地址等关键信息。

## 页面说明

### 链列表表格

| 字段 | 说明 |
|------|------|
| 链名称 | 区块链网络名称（如 Ethereum、Tron、Base 等） |
| RPC | RPC 节点 URL |
| 提币地址 | 该链上用于发起提币的热钱包地址 |
| 状态 | 正常（绿色）/ 停用（红色） |
| 创建时间 | 创建时间 |
| 更新时间 | 最后更新时间 |
| 操作 | 编辑按钮 |

## 操作指南

### 添加链

<Steps>
  <Step title="点击「添加」按钮">
    页面右上角点击「添加」按钮。
  </Step>
  <Step title="填写链信息">
    - **链名称**（必填）：如 Ethereum、Tron、Arbitrum 等
    - **RPC**（必填）：区块链 RPC 节点 URL
    - **提币地址**（可选）：该链上的提币热钱包地址
  </Step>
  <Step title="确认提交">
    点击确定完成创建。
  </Step>
</Steps>

### 编辑链

<Steps>
  <Step title="点击「编辑」按钮">
    在操作列点击「编辑」按钮。
  </Step>
  <Step title="修改信息">
    可修改链名称、RPC、提币地址和状态。
  </Step>
  <Step title="保存">
    点击确定保存修改。
  </Step>
</Steps>

## 权限要求

| 功能 | 超管(1) | 其他角色 |
|------|:------:|:------:|
| 查看链列表 | ✅ | ❌ |
| 添加链 | ✅ | ❌ |
| 编辑链 | ✅ | ❌ |

## 注意事项

<Warning>
  **停用链** 后，该链上的所有充值地址将无法接收新交易，相关币种的充提功能也会受到影响。请谨慎操作。
</Warning>

<Warning>
  修改 RPC 地址前请确保新的 RPC 节点可用，否则可能导致链上数据查询和交易处理异常。
</Warning>

<Tip>
  建议使用高可用的 RPC 节点服务（如 Alchemy、Infura 等），避免因 RPC 不稳定导致交易处理延迟。
</Tip>
