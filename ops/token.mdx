币种管理模块用于管理系统支持的加密货币代币。每个币种关联到一条链，包含合约地址和充提控制开关。

## 页面说明

### 币种列表表格

| 字段 | 说明 |
|------|------|
| 币种名称 | 代币内部名称（如 ETH_USDT、TRON_USDT 等） |
| 所属链 | 该代币部署的区块链网络 |
| 合约地址 | 代币智能合约地址 |
| 是否可充值 | 是（绿色）/ 否（红色） |
| 是否可提现 | 是（绿色）/ 否（红色） |
| 创建时间 | 创建时间 |
| 更新时间 | 最后更新时间 |
| 操作 | 编辑按钮 |

## 操作指南

### 添加币种

<Steps>
  <Step title="点击「添加」按钮">
    页面右上角点击「添加」按钮。
  </Step>
  <Step title="填写币种信息">
    - **币种名称**（必填）：如 ETH_USDT
    - **币种**（必填）：代币符号，如 USDT
    - **所属链**（必填）：选择该代币所在的区块链
    - **合约地址**（必填）：代币智能合约地址
    - **可充值**：是否允许充值
    - **可提现**：是否允许提现
  </Step>
  <Step title="确认提交">
    点击确定完成创建。
  </Step>
</Steps>

### 编辑币种

<Steps>
  <Step title="点击「编辑」按钮">
    在操作列点击「编辑」按钮。
  </Step>
  <Step title="修改信息">
    可修改币种名称、币种符号、充提开关。所属链和合约地址创建后不可修改。
  </Step>
  <Step title="保存">
    点击确定保存修改。
  </Step>
</Steps>

## 权限要求

| 功能 | 超管(1) | 其他角色 |
|------|:------:|:------:|
| 查看币种列表 | ✅ | ❌ |
| 添加币种 | ✅ | ❌ |
| 编辑币种 | ✅ | ❌ |

## 注意事项

<Warning>
  关闭「可充值」或「可提现」开关后，用户将无法使用该币种进行对应操作。常用于紧急暂停某个币种的充提。
</Warning>

<Warning>
  合约地址一旦填写后不可修改。请确保填写的是正确的、已验证的合约地址。
</Warning>

<Tip>
  币种名称建议使用 `{链名}_{币种}` 的格式（如 ETH_USDT、TRON_USDT），方便在其他模块中快速识别。
</Tip>
