钱包管理模块用于管理项目级别的钱包。每个 App（项目）可以有一个或多个钱包，钱包是资金归集和操作的核心单元。

## 页面说明

### 钱包列表表格

| 字段 | 说明 |
|------|------|
| wallet_id | 钱包唯一标识（可复制） |
| 描述 | 钱包用途说明 |
| 所属 App | 该钱包归属的项目 |
| 状态 | 正常（绿色）/ 停用（红色） |
| 创建时间 | 钱包创建时间 |
| 更新时间 | 最后更新时间 |
| 操作 | 编辑按钮 |

## 操作指南

### 添加钱包

<Steps>
  <Step title="点击「添加」按钮">
    页面右上角点击「添加」按钮。
  </Step>
  <Step title="填写钱包信息">
    - **wallet_id**（必填）：钱包唯一标识
    - **所属 App**（必填）：选择归属的项目
    - **描述**（必填）：钱包用途说明
  </Step>
  <Step title="确认提交">
    点击确定完成创建。
  </Step>
</Steps>

### 编辑钱包

<Steps>
  <Step title="点击「编辑」按钮">
    在操作列点击「编辑」按钮。
  </Step>
  <Step title="修改信息">
    可修改描述和状态。所属 App 创建后不可修改。
  </Step>
  <Step title="保存">
    点击确定保存修改。
  </Step>
</Steps>

## 权限要求

| 功能 | 超管(1) | 其他角色 |
|------|:------:|:------:|
| 查看钱包列表 | ✅ | ❌ |
| 添加钱包 | ✅ | ❌ |
| 编辑钱包 | ✅ | ❌ |

## 注意事项

<Warning>
  钱包停用后，关联的充值地址和交易功能将受到影响。请确认后再操作。
</Warning>

<Tip>
  钱包的 `wallet_id` 在系统中被广泛引用（充值记录、划转等模块均显示钱包名称），建议使用有意义的描述方便识别。
</Tip>
