CEX 提币记录模块用于追踪从中心化交易所提币到链上地址的所有记录。

## 页面说明

### 统计卡片

表格上方展示当前筛选条件下的汇总数据：

| 指标 | 说明 |
|------|------|
| USDT 划转总额 | 筛选条件下 USDT 提币的总金额 |
| USDC 划转总额 | 筛选条件下 USDC 提币的总金额 |

### 提币记录表格

| 字段 | 说明 |
|------|------|
| 所属链 | 提币使用的区块链网络 |
| 提币数量 | 提币的代币数量 |
| 币种 | 代币类型和名称（如 USDT (ETH_USDT)） |
| 提币来源 | 资金来源（如 CEX 名称） |
| 到账地址 | 提币目标链上地址（可复制、可跳转浏览器） |
| 提币时间 | 提币操作时间 |
| 所属钱包 | 提币归属的项目钱包 |
| 请求 ID | 提币请求的唯一标识 |
| 交易 ID | 系统内部交易标识 |

### 筛选条件

| 筛选项 | 类型 | 说明 |
|--------|------|------|
| 所属链 | 下拉框 | 按区块链网络筛选 |
| 币种 | 下拉框 | 按代币种类筛选 |
| 所属钱包 | 下拉框 | 按项目钱包筛选 |
| 起止时间 | 日期范围 | 选择查询的时间范围 |

## 操作指南

<Steps>
  <Step title="设置筛选条件">
    选择链、币种、钱包和时间范围来缩小查询范围。
  </Step>
  <Step title="查看汇总数据">
    注意表格上方的 USDT/USDC 划转总额统计卡片。
  </Step>
  <Step title="查看提币详情">
    在表格中查看每笔提币记录，点击地址可复制或跳转区块链浏览器确认。
  </Step>
</Steps>

|------|:------:|:------:|:------:|:------:|:------:|
| 查看 CEX 提币记录 | ✅ | ❌  | ❌  | ❌  | ❌  |

## 注意事项

<Note>
  CEX 提币记录为只读数据，用于审计和追踪资金从 CEX 流出的记录。如需发起提币操作，请通过 CEX 平台进行。
</Note>

<Tip>
  结合「对账」模块使用，可以验证 CEX 提币金额是否与链上到账一致。
</Tip>
