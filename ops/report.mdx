报表模块提供账单导出功能，可将交易明细导出为 Excel 文件，用于财务对账和内部审计。

## 页面说明

页面以卡片形式展示可导出的报表类型：

<CardGroup cols={2}>

## 操作指南

### 导出账单

<Steps>
  <Step title="点击「导出」按钮">
    在「账单」卡片下方点击「导出」按钮，打开导出弹窗。
  </Step>
  <Step title="选择项目范围">
    - **超管**：从下拉框选择特定钱包或选择「全部钱包」
    - **普通管理员**：可选「当前项目」或「所有项目」
  </Step>
  <Step title="选择日期范围">
    系统默认选择最近 30 天。点击日期选择器自定义范围。
  </Step>
  <Step title="确认导出">
    点击「导出」按钮，系统会生成 Excel 文件并自动下载。
  </Step>
</Steps>

### 导出文件字段说明

| 字段 | 说明 |
|------|------|
| 序号 | 行号 |
| 交易创建时间 | 交易创建的日期时间 |
| 交易入账时间 | 交易实际入账的日期时间 |
| 交易类型 | 网关代收 / 结算 / 网关代付 / 划转 / 增加余额 / 扣减余额 |
| 汇率 | 交易时的汇率 |
| 入账金额 | 入账方向的金额（正数） |
| 支出金额 | 支出方向的金额（负数） |
| 交易币种 | USD |
| 手续费 | 该笔交易的手续费 |
| 手续费币种 | 手续费结算币种 |
| 手续费类型 | App / User（仅 Payout 管理员可见） |
| 账户余额 | 该笔交易后的账户余额 |
| 交易 ID | 系统内部交易唯一标识 |
| 所属项目 | 交易归属的项目 |

<Note>
  非 Payout 管理员/运营角色导出的 Excel 中不包含「手续费类型」列。
</Note>

## 权限要求

| 功能 | 超管 | 管理员(Payin) | Operator(Payin) | 管理员(Payout) | Operator(Payout) |
|------|:------:|:------:|:------:|:------:|:------:|
| 导出账单 | ✅ 全部 | ✅ 本项目 | ✅ 本项目 | ✅ 本项目 | ✅ 本项目 |

## 注意事项

<Warning>
  选择「所有项目」导出时，必须指定日期范围，避免数据量过大导致导出失败。
</Warning>

<Tip>
  导出的文件名格式为 `statement_{项目名}_{起始日期}_{结束日期}.xlsx`，方便按项目和时间归档。
</Tip>
