## 页面说明

### 商户余额卡片

页面顶部展示当前可用余额：

| 指标 | 说明 |
|------|------|
| 可用余额 | 当前可发起结算的 NUSD 余额 |

### 结算历史表格

| 字段 | 说明 |
|------|------|
| 银行名称 | 收款银行名称（通过银行账户记录匹配） |
| 银行账户 | 结算到账的银行账号 |
| 项目 | 所属项目（超管可见） |
| 订单号 | 结算交易唯一标识 |
| 结算金额(NUSD) | 申请结算的 NUSD 数量 |
| 服务费(NUSD) | 结算手续费 |
| 汇率 | 结算时使用的汇率 |
| 实际到账 | 扣除手续费后实际到账的法币金额 |
| 结算币种 | 目标法币（USD） |
| 结算时间 | 结算发起时间 |
| 状态 | 结算单当前状态 |
| 操作 | 审核/取消按钮 |

### 筛选条件

| 筛选项 | 类型 | 说明 |
|--------|------|------|
| 银行账户 | 输入框 | 按银行账户号搜索 |
| 项目 | 下拉框 | 超管可按项目筛选 |

## 操作指南

### 发起结算（管理员(Payin)角色）

<Steps>
  <Step title="点击「结算」按钮">
    页面右上角点击「结算」按钮，打开结算弹窗。
  </Step>
  <Step title="选择结算币种">
    默认为 USD，目前仅支持 USD。
  </Step>
  <Step title="选择银行账户">
    从已审核通过的银行账户列表中选择收款账户。
  </Step>
  <Step title="输入结算金额">
    输入要结算的 NUSD 数量。可点击「可用余额」数字一键填充全部余额。
  </Step>
  <Step title="获取验证码">
    点击「获取验证码」，系统发送验证码到注册邮箱。
  </Step>
  <Step title="输入验证码并提交">
    填入验证码后点击确定，结算申请进入「待确认」状态。
  </Step>
</Steps>

### 审核结算（超管）

<Steps>
  <Step title="找到待确认的结算单">
    在结算历史表格中，找到状态为「待确认」的记录。
  </Step>
  <Step title="点击「审核」按钮">
    点击操作列的「审核」按钮，弹出审核弹窗。
  </Step>
  <Step title="选择审核结果">
    选择「审核通过」或「驳回」，点击确定。
  </Step>
</Steps>

### 取消结算（管理员角色）

在结算历史中找到「待确认」状态的记录，点击「取消」按钮，确认后即可取消该笔结算。

## 注意事项

<Warning>
  发起结算需要**邮箱验证码**二次确认，确保资金安全。验证码 60 秒内有效。
</Warning>

<Warning>
  结算前请确保已在「银行账户」模块添加并通过审核的银行账户，否则无法选择收款账户。
</Warning>

<Tip>
  结算金额不要超过可用余额。可点击余额数字一键填充。
</Tip>
