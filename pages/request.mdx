## 身份验证

为确保您加密资产的安全访问，您需要在请求头中提供 API Key、随机数和 API 签名：

``` javascript
headers = {
  "Biz-Api-Key": API_KEY,
  "Biz-Api-Nonce": Nonce,
  "Biz-Api-Signature": API_SIGNATURE
}
```

- Biz-Api-Key：用于请求的 API Key。
- Biz-Api-Nonce：随机数。随机数是以毫秒为单位的当前 Unix 时间戳。
- Biz-Api-Signature：API 签名。要了解如何计算 API 签名，请参阅计算 [API 签名](/pages/encrypt)。
