## 接口规则

**传输方式：** https

| 项目         | 内容                                                                 |
|--------------|----------------------------------------------------------------------|
| **请求地址** | 域名 + 接口地址                                                       |
| **请求头**   | Content-Type: application/x-www-form-urlencoded                       |
| **加密算法** | RSA，参考 [加密说明](/encode) |
| **响应头**   | Content-Type: application/json; charset=utf-8                         |
| **响应状态码** | 0 表示处理**成功**，非 0 表示请求错误或系统异常                       |

## 请求参数统一格式

| 参数         | 是否必传 | 说明                                     |
|--------------|-----|-----------------------------------------|
| **app_id** | 必传 | string类型，商户id，暂时由我们提供                          |
| **data**   | 非必传 | string类型，<code>json格式的请求数据</code>加密之后的字符串，加密方法参考[加密说明](/encode) |


## 返回结果统一格式

| 参数          | 说明                                     |
|--------------|----------------------------------------------|
| **data**     | string类型，<code>json格式的数据</code>加密之后的字符串，解密方法参考[解密说明](/decode)|

### 响应结果data解密后格式

| 参数          | 类型        |    说明                                     |
|--------------|-------------|--------------------------------------------|
| **code**     | string      |   状态码，例：100002                         |
| **msg**      | string      |   响应结果说明，例：请求参数错误                     |
| **data**     | string      |   具体响应数据，数据结构定义见各接口                         |
