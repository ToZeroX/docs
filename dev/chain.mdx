## USDT 支持的公链

- TRON (TRON_USDT)
- Ethereum (ETH_USDT)
- Arbitrum (ARBITRUM_USDT)
- Base (BASE_USDT)
- Polygon (MATIC_USDT)
- BNB Smart Chain (BSC_USDT)

``` javascript
[
    "TRON_USDT",
    "ETH_USDT",
    "ARBITRUM_USDT",
    "BASE_USDT",
    "MATIC_USDT",
    "BSC_USDT",
]
```

## USDC 支持的公链

- Ethereum (ETH_USDC)
- Arbitrum (ARBITRUM_USDCOIN)
- Base (BASE_USDC)
- Polygon (MATIC_USDC2)
- BNB Smart Chain (BSC_USDC)

``` javascript
[
    "ETH_USDC",
    "ARBITRUM_USDCOIN",
    "BASE_USDC",
    "MATIC_USDC2",
    "BSC_USDC",
]
```

## NUSD 支持的公链

- Base (BASE_NUSD)

``` javascript
[
    "BASE_NUSD",
]
```

<Note>
  在收到webhook充值通知时，需要判断一下消息中的token_id，只有在上面的数组中包含该token_id时，才是正确的充值信息，如果token_id不属于上面三个数组，则忽略该条消息。
</Note>

## Payout 支持情况

<Note>
**充值**：用户向由 API 生成的地址转账，资金进入钱包余额  
**Payout**：通过 API 从钱包余额提取资金到指定地址
</Note>

| 链 | 代币 | Payin | Payout | 预估确认时间 |
|----------|--------|:------:|:------:|------|
| Ethereum | USDT, USDC | ✅ | ✅ | ~2 分钟 |
| Tron | USDT | ✅ | ❌ | ~1 分钟 |
| Arbitrum | USDT, USDC | ✅ | ✅ | ~30 秒 |
| Polygon | USDT, USDC | ✅ | ✅ | ~30 秒 |
| BSC | USDT, USDC | ✅ | ✅ | ~30 秒 |
| Base | USDT, USDC | ✅ | ✅ | ~30 秒 |

<Warning>
  **TRON Payout 暂不支持** — TRON 网络目前仅支持 Payin（充值收款），Payout 功能敬请期待。
</Warning>

## 链选择建议

| 使用场景 | 推荐链 | 原因 |
|----------|--------|------|
| 低手续费交易 | MATIC、BSC | Gas 费用较低 |
| 高频小额交易 | Base、Arbitrum | L2 网络，快速确认 |
| 大额安全交易 | Ethereum | 最去中心化，最安全 |
| DeFi 生态丰富 | Ethereum、BSC | 应用生态完善 |
