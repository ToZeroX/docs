NUSDpay 平台的费率按项目独立配置，支持灵活的费率方案。本文档说明各业务环节的费率模型和计算方式。

## 费率总览

| 业务环节 | 费率类型 | 说明 |
|---------|---------|------|
| **Payin（收款）** | 百分比手续费 | 按充值金额的百分比收取 |
| **Payout（付款）** | 百分比手续费 | 按付款金额的百分比收取 |
| **结算（Settlement）** | 固定费用 + OTC 费率 | 按笔固定费用 + 结算金额的百分比 |
| **划转（Transfer）** | 免费 | 项目间 NUSD 调拨不收取手续费 |

<Info>
  以上费率按项目独立配置，不同项目可能有不同的费率方案。具体费率以签约协议为准。可通过管理后台 **Fee Station** 或 [查询平台费用](/api-reference/endpoint/fees) 接口查看当前项目的费率配置。
</Info>

## Payin 手续费

终端用户向商户收款地址付款时，NUSDpay 按充值金额收取手续费，扣除后以 NUSD 入账。

```
商户实际到账(NUSD) = 充值金额 × (1 - Payin费率) × 汇率
```

| 参数 | 说明 | 示例 |
|------|------|------|
| 充值金额 | 终端用户链上转入的原始金额 | 1000 USDT |
| Payin 费率 | 按项目配置的百分比费率 | 1.00% |
| 汇率 | 充值代币 → NUSD 的实时汇率（如配置了汇率转换） | 1.0000 |
| 实际到账 | 商户钱包收到的 NUSD | 990 NUSD |

### 汇率转换

Payin 支持两种模式：

- **无汇率转换**：充值代币按 1:1 直接入账为 NUSD（适用于 NUSD 代币充值）
- **有汇率转换**：充值代币按实时市场汇率换算后入账为 NUSD（适用于 USDT/USDC 充值）

<Tip>
  汇率转换是否启用按币种独立配置，可在 Fee Station 中查看。
</Tip>

## Payout 手续费

商户从 NUSD 余额向终端用户钱包地址发起链上付款时，手续费有两种承担模式：

### App 付费模式（商户承担）

```
终端用户到账 = 支付金额 × 汇率
手续费 = 支付金额 × Payout费率（从商户余额额外扣除）
```

### User 付费模式（终端用户承担）

```
终端用户到账 = 支付金额 × (1 - Payout费率) × 汇率
```

### Payout 汇率转换

Payout 时 NUSD 需要换算为链上代币（USDT/USDC），系统按实时汇率进行转换。

## 结算手续费

商户将 NUSD 结算为 USD 法币时：

```
实际到账(USD) = 结算金额(NUSD) × 汇率 - 固定费用 - (结算金额 × OTC费率)
```

| 参数 | 说明 |
|------|------|
| 固定费用 | 每笔结算的固定手续费（按项目配置） |
| OTC 费率 | 结算金额的百分比费率（按项目配置） |

## 限额规则

| 限制类型 | 说明 |
|---------|------|
| 最小提币数量 | 单笔 Payout 的最低金额，按币种配置 |
| 最大提币数量 | 单笔 Payout 的最高金额，按币种配置 |

<Warning>
  低于最小提币数量或超过最大提币数量的 Payout 请求将被拒绝。具体限额可在 Fee Station 中查看。
</Warning>

## Gas 费

链上交易的 Gas 费由 NUSDpay 平台垫付，已包含在服务费率中，商户无需额外关注。

## 查询当前费率

商户可通过以下方式查看当前项目的费率配置：

- **管理后台**：进入 **Fee Station** 页面，查看各币种的费率、汇率转换开关和限额配置
- **API 接口**：调用 [查询平台费用](/api-reference/endpoint/fees) 接口获取程序化的费率数据
