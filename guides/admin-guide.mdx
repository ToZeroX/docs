## 角色与权限

NUSDpay 管理后台采用基于角色的权限控制（RBAC），不同角色可访问的功能模块不同。

### 角色说明

| 角色 | 定位 |
|------|------|
| 超管 | 平台全局管理，可访问所有功能和所有项目数据 |
| 管理员(Payin) | Payin 收单业务的项目管理，可发起结算、管理工单等 |
| Operator(Payin) | Payin 侧只读权限，仅查看数据和记录 |
| 管理员(Payout) | Payout 代付业务的项目管理 |
| Operator(Payout) | Payout 侧只读权限，仅查看数据和记录 |

## 常用操作

### 查询交易

1. 进入 **交易记录** 页面
2. 使用筛选条件：交易类型（充值/支付）、状态、日期范围、交易 ID
3. 点击具体交易查看详情，包含链上交易哈希、金额、手续费、状态变更记录

### 查看余额

- **项目余额**：页面顶部显示当前项目的可用余额
- **全局余额**：超管可在结算页面查看所有项目的汇总余额

### 发起结算

详见 [结算方案](/guides/settlement)。

### 发起划转

详见 [划转方案](/guides/transfer)。

### 导出报表

详见 [对账与报表](/ops/reconciliation)。

### 添加团队成员

1. 进入 **团队管理** 页面
2. 点击 **添加成员**，填写邮箱和角色
3. 新成员收到邀请邮件后完成注册

<Warning>
  请遵循最小权限原则分配角色。日常运营人员使用「运营/财务」角色即可，避免分配超管权限。
</Warning>

## 安全操作

以下敏感操作需要验证码二次确认：
- 发起结算
- 添加/修改银行账户
- 修改团队成员权限
