结算（Settlement）允许商户将 NUSD 钱包余额兑换为 USD 法币，汇入商户绑定的银行账户。这是资金从加密货币回到传统金融体系的出口。

## 操作步骤

<Steps>
  <Step title="绑定银行账户">
    商户需预先在管理后台绑定银行账户信息（银行名称、账户名、账号），经平台审核通过后方可用于结算。
  </Step>

  <Step title="发起结算">
    在管理后台 **结算** 页面，选择已审核通过的银行账户，输入结算金额（NUSD），并通过验证码验证身份。

    <Info>
      结算币种固定为 USD。系统按实时汇率将 NUSD 换算为 USD，扣除服务费后为实际到账金额。
    </Info>
  </Step>

  <Step title="等待审核">
    结算申请提交后进入待审核状态，NUSD 资金被冻结。超管在管理后台审核（通过/驳回）。
  </Step>

  <Step title="法币到账">
    审核通过后，NUSDpay 发起法币汇款到商户银行账户。到账时间取决于银行处理速度。
  </Step>
</Steps>

## 结算金额计算

```
实际到账(USD) = 结算金额(NUSD) × 汇率 - 服务费(NUSD)
```

| 字段 | 说明 |
|------|------|
| 结算金额(NUSD) | 商户申请结算的 NUSD 数量 |
| 汇率 | NUSD → USD 的实时汇率 |
| 服务费(NUSD) | 按项目配置的结算费率计算 |
| 实际到账(USD) | 商户银行账户实际收到的 USD 金额 |

<Tip>
  结算记录中可查看每笔交易的实际汇率和服务费明细。
</Tip>

## 结算状态

| 状态 | 说明 |
|------|------|
| 待确认 | 已提交，等待超管审核 |
| 审核通过 | 审核已通过，正在处理汇款 |
| 已取消 | 商户主动取消结算 |
| 已驳回 | 超管审核驳回，NUSD 冻结释放 |
| 已完成 | 法币已汇出，结算完成 |
| 交易失败 | 汇款处理失败 |

## 权限说明

| 角色 | 可执行操作 |
|------|-----------|
| 超管 | 查看所有项目结算记录、审核（通过/驳回） |
| 项目管理员 | 发起结算、取消自己的待审核结算 |
| 财务 / Operator(Payout) | 查看结算历史记录 |

<Warning>
  结算审核通过后不可撤销。请在提交前仔细核对银行账户和金额。
</Warning>
